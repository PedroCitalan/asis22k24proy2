Funciones de los Botones

  • Botón Guardar: Este boton sirve para ingresar los datos dentro de un formulario que va hacia la base de datos y muestra la informacion en el datagriedview.
  • Botón Modificar: Funciona para modificar los datos que se ingresaron por medio del id, para modificar los campos que se establecen en el formulario.
  • Botón Eliminar: Su funcion es eliminar los datos de pedido que ya no son necesarios o que fueron ya entregados.
  • Botón Actualizar: Este botón, refresca los datos que han sido modificados o almacenados en la base de datos, para verse reflejados en el dataGridview
  • Botón Ver Reporte: Este nos traslada a otra ventana llamada visualizar, en el que podremos observar el archivo del reporte previamente seleccionado,con los datos que se ingresaron.
